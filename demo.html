Zoom in

Use the scale property to create a zoom effect. scale: .5 make the element zoom from half size. It is recommended to combine this with opacity: 0

const options = {
	style: {
		scale: .5,
		opacity: 0,
	}
}
new FancyLoad(elements, options);
